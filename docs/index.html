🏗 System Architecture

This tool is a browser-based port of a Python production planning AI agent. The full Python version reads/writes Microsoft Excel using pandas and openpyxl. This web version replicates all 9 pipeline steps in pure JavaScript so the system runs without any installation.

StepFunctionDescription
1load_data()Load all data from Excel / browser state
2forecast_demand()Aggregate orders by flavor, sort by priority + due date
3convert_to_batches()Convert bottles → liters → batch sizes
4check_materials()BOM explosion vs. inventory stock levels
5–6assign_tanks_and_schedule()Greedy EDD tank assignment + fermentation dates
7schedule_bottling()Bottling slot assignment with overload detection
8detect_bottlenecks()Consolidate all alerts and insights
9write_output()Render results to Dashboard, Schedule, Alerts tabs
